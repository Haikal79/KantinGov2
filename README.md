# KantinGo (Simulasi MPL 2.0)

KantinGo adalah aplikasi simulasi untuk pemesanan makanan di kantin kampus.

## Fitur
- Lihat menu makanan
- Buat pesanan
- Tampilkan daftar pesanan

## Teknologi
- Node.js (Express)
- React.js (Frontend)
